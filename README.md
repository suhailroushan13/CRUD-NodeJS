# NodeJS RestFull CRUD API Project | No Framework or Library
> This is vanilla Node.js CRUD API movie project.

## Routes
```bash
GET      /api/movies
POST     /api/movies
GET      /api/movies/:id
PUT      /api/movies/:id
DELETE   /api/movies/:id

```

## Installation
```bash
git clone git@github.com:suhailroushan13/CRUD-NodeJS.git
# Install dependencies
npm install
# Run App
npm start
```
